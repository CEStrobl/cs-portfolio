Python Right Triangle Calculator

Class: CSCI 301 - Survey of Scripting Languages

Grade: 100/100

Language(s): Python

About the Project

This Python program is a utility for calculating key properties of right triangles. It uses a custom RightTriangle class to encapsulate the triangle's properties, including its two shorter sides, and offers methods to compute the hypotenuse and surface area based on user-provided dimensions.

The program features intuitive getter and setter methods for modifying and retrieving the lengths of the triangle's sides. Using the Pythagorean theorem, it dynamically calculates the hypotenuse and determines the surface area with a simple formula. This approach keeps the program modular and easy to extend.

Designed for simplicity and precision, the program prints the side lengths, hypotenuse, and surface area of the triangle. It's ideal for educational purposes or as a quick computational tool for geometry-related tasks.

Fig. 1: Example output with input of 3 and 4


How to Run

To run the calculation, execute the Python script using the following command:

python3 RightTriangle.py

After executing the program, it initializes a right triangle with predefined side lengths and prints the lengths of both sides. It then calculates and displays the triangle's hypotenuse and surface area based on the input values.
