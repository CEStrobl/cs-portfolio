Python Web Scraper

CSCI 301 - Grade: 100

This project involves developing a Python web scraper to efficiently extract specific job-related data from a website, targeting the HTML tags h2, h3, and p. Using Python libraries like BeautifulSoup, the tool identifies and retrieves the job title, company name, and location, then displays this information in the command line. This structured approach to web scraping allows for quick parsing and organized output, making it easier to analyze job postings. Through its direct, targeted parsing, this tool demonstrates how automation can streamline data gathering from the web in a systematic, repeatable way.
