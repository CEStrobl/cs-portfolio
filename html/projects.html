Academic Projects

This section highlights my school projects, showcasing my programming and problem-solving skills. I applied various programming languages to solve complex challenges, focusing on efficiency and usability. These experiences demonstrate my ability to think critically and adapt to new problems.


Python Web Scraper

CSCI 301 - Grade: 100

This project involves developing a Python web scraper to efficiently extract specific job-related data from a website, targeting the HTML tags h2, h3, and p. Using Python libraries like BeautifulSoup, the tool identifies and retrieves the job title, company name, and location, then displays this information in the command line. This structured approach to web scraping allows for quick parsing and organized output, making it easier to analyze job postings. Through its direct, targeted parsing, this tool demonstrates how automation can streamline data gathering from the web in a systematic, repeatable way.

Java Card Game Collection

CSCI 325 - Grade: 100

This project is a Java-based card game collection developed collaboratively, with each team member responsible for creating a unique game. I contributed "Twenty-One," also known as Blackjack, as an interactive command-line game. The game follows standard Blackjack rules, allowing users to place bets, receive cards, and choose whether to hit or stand. Through structured classes and methods, "Twenty-One" simulates a real gameplay experience by handling card dealing, scoring, and win-loss conditions. This project emphasizes object-oriented programming principles and demonstrates effective teamwork in building a cohesive, multi-game experience within a shared codebase.
